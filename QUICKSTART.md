# Torrent Searcher - Быстрый Старт

## Установка и Запуск

### 1. Установите зависимости
```bash
npm install
```

### 2. Соберите проект
```bash
npm run build
```

### 3. Запустите сервер
```bash
npm start
```

Приложение будет доступно по адресу: **http://localhost:3000**

---

## Использование

### Поиск торрент файлов
1. Откройте http://localhost:3000
2. Введите название файла в поисковую строку
3. Нажмите кнопку Search
4. Результаты покажут список найденных торрентов с magnet-ссылками
5. Кликните на "Magnet Link" для скачивания

### API для программы minglegregated

**Поиск:**
```
GET /api/search?q=<поисковый запрос>&page=<страница>&limit=<количество>

Пример:
curl "http://localhost:3000/api/search?q=test"
```

**Проверка здоровья:**
```
GET /api/health
```

**Проверка статуса тректоров:**
```
GET /api/trackers
```

## Конфигурация

### Изменение порта
```bash
PORT=4000 npm start
```

### Настройка прокси (для обхода блокировок)
```bash
export HTTP_PROXY=http://proxy.example.com:8080
export http_proxy=http://proxy.example.com:8080
npm start
```

---

## Добавление нового трекера

Редактируйте `server/src/config/trackers.ts`:

```typescript
{
  name: 'Ваш Трекер',
  baseUrl: 'https://tracker.example.com',
  searchPath: '/search/',
  enabled: true,
  requiresProxy: true,
}
```
